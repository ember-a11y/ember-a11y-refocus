# How To Contribute

## Installation

* `git clone <repository-url>`
* `cd ember-a11y-refocus`
* `pnpm install`

## Linting

* `pnpm lint`
* `pnpm lint:fix`

## Building

* `pnpm build`

## Running tests

* `pnpm test` – Builds and runs the test suite
* `pnpm start` – Starts the development server in watch mode

Visit the demo application at [http://localhost:4200](http://localhost:4200).

For more information on developing v2 addons, visit the [Ember CLI Guides](https://cli.emberjs.com/release/).
